# Contributing to Tantra-Trace

We welcome contributions to the Atulya Tantra ecosystem! 

## How to Contribute
1. **Fork the repository.**
2. **Create a new branch** (`git checkout -b feature/amazing-feature`).
3. **Commit your changes** (`git commit -m 'Add amazing feature'`).
4. **Push to the branch** (`git push origin feature/amazing-feature`).
5. **Open a Pull Request.**

## Standards
- Follow the PEP 8 style guide for Python.
- Ensure all logic aligns with the "Saner Systems" philosophy (Relief > Brilliance).
- Add tests for new features.

Thank you for helping us build saner AI systems!
