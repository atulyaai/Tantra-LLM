CPU-first language model with **auto-scaling vocabulary**, **sparse mixture-of-strands routing**, and **curriculum learning** from 20 GB of diverse training data. Runs entirely on CPU — no GPU required.

## Architecture

```text
Token IDs → Embedding → [Mesh₁ → … → Meshₙ] → Norm → LM Head
                           │
                    ┌──────┴──────┐
                    │  Router     │
                    │  (top-k)    │
                    └──────┬──────┘
                    ┌──────┴──────┐
                    │  Strands    │  ← DNA weight generation
                    │  (SSM/Attn) │     (low-rank U@V)
                    └─────────────┘
```

Each token is routed to only the **top-k** most relevant Strands out of N total — true sparse execution (Switch Transformer style), not dense masking. This gives **N/k × compute savings** (e.g., 100 strands, top-3 = 33×).

## Features

- **Auto-scaling vocabulary** — BPE tokenizer grows from 8K → 256K tokens on demand when fill exceeds 95%. Never hits unknown token issues.
- **Curriculum learning** — 8-stage training from 57 Q&A pairs to 20 GB across samples, agentic, factual, code, reasoning, translation, general, and math data.
- **Dynamic strand allocation** — strands and layers grow at runtime. Router expands in-place without re-initialization.
- **DNA weight generation** — All strand weights are generated by a shared low-rank Genome (U@V with configurable rank). Adding strands adds ~0 params per strand for the Genome.
- **Attention + SSM strands** — Each layer can use causal SSM (gated recurrence) or full attention. Configurable per-layer.
- **Memory cortex** — Retrieval-augmented memory store with dense index and top-k lookup.

## Train

```powershell
$env:KMP_DUPLICATE_LIB_OK="TRUE"; python npdna\train_npdna_v3.py
```

**Curriculum** (8 stages, ~9100 steps, ~2h on CPU):

| Stage | Dataset | Size | Steps |
|---|---|---|---|
| 0 | samples | 0 MB | 0→500 |
| 1 | +agentic | 151 MB | 500→800 |
| 2 | +factual | 220 MB | 800→1150 |
| 3 | +code | 556 MB | 1150→1650 |
| 4 | +reasoning | 559 MB | 1650→2150 |
| 5 | +translation | 1.8 GB | 2150→3250 |
| 6 | +general | 7.6 GB | 3250→5250 |
| 7 | +math (all) | 20 GB | 5250→9100 |

Resume automatically on re-run — picks up at exact step + stage from checkpoint.

## Generate

```python
from npdna import NpDnaCore

core = NpDnaCore.load("model/npdna_v3/best")
text = core.generate("What is gravity?", max_tokens=100,
                     temperature=0.3, top_k=30,
                     repetition_penalty=1.2)
print(text)
```

## Generation Parameters

| Param | Value | Effect |
|---|---|---|
| temperature | 0.3 | Low = focused, deterministic |
| top_k | 30 | Limit to 30 most likely tokens |
| top_p | 0.85 | Nucleus sampling |
| repetition_penalty | 1.2 | Discourage repeats |
