NP-DNA is a CPU-first, auto-scaling language model that grows its vocabulary, strands, and layers on demand.

```text
NP-DNA = Genome (low-rank U@V) + Sparse MoE Mesh + Strands (SSM/Attention) + BPE Tokenizer
```

The current repository is a working prototype with curriculum learning from 20 GB of diverse data, BPE tokenization, sparse mixture-of-strands routing, and DNA-weight generation — all running entirely on CPU.

## New

- **Dynamic vocabulary** — BPE tokenizer grows from 8K → 256K tokens on demand when fill exceeds 95%. Byte-fallback ensures zero unknown tokens.
- **Sparse MoE routing** — Each token activates only top-k strands out of N total. True sparse execution (Switch Transformer style), not dense masking. Gives N/k × compute savings.
- **DNA weight generation** — All strand weights generated by a shared low-rank Genome (U@V, R=32). Adding strands adds zero Genome parameters.
- **Curriculum learning** — 8-stage training from 57 Q&A pairs to 20 GB across samples, agentic, factual, code, reasoning, translation, general, and math.
- **Auto-scaling** — Strands, layers, and vocab all grow at runtime via `add_strand()`, `add_layer()`, `resize_embeddings()`.
- **Attention + SSM strands** — Configurable per-layer. Use causal SSM (gated recurrence) or full self-attention.

```mermaid
flowchart TD
    I["Token IDs"] --> E["Embedding<br/>(auto-resizing)"]

    N1 --> M2["Mesh Layer 2<br/>Router → Strands"]
    M2 --> N2["LayerNorm"]
    N2 --> CX["Cortex<br/>memory augmentation"]

    FN --> H["LM Head"]

    G["Genome<br/>(U@V)"]
    G -.-> M1
    G -.-> M2

    sub_graph ["Each Mesh Layer"]
    R["Router<br/>(top-k)"] --> A["Assignment"]
    A --> S1["Strand 1"]
    A --> S2["Strand 2"]
    A --> S3["... Strand N"]
    S1 --> W["Weighted Sum"]
    S2 --> W
    S3 --> W
```

**Curriculum** (8 stages, ~9100 steps, ~2h on CPU):

| Stage | Dataset | Size | Steps |
|---|---|---|---|
| 0 | samples | 0 MB | 0→500 |
| 1 | +agentic | 151 MB | 500→800 |

| 6 | +general | 7.6 GB | 3250→5250 |

## API (Python)

```python
from npdna import NpDnaCore, NpDnaConfig

# Create fresh model
core = NpDnaCore.from_config("nano")
ids = core.encode("Hello, world!")
logits, loss = core.model(torch.tensor([ids]))
text = core.generate("Hello", max_tokens=50)

# Load checkpoint
core = NpDnaCore.load("model/npdna_v3/best")
text = core.generate("What is gravity?", max_tokens=100,
                     temperature=0.3, top_k=30, repetition_penalty=1.2)
print(text)

# Auto-growing
core.model.grow_strands(count=2)  # adds strands to every layer
core.model.add_layer(name="reasoning", num_strands=8)  # new layer
core.tokenizer.encode("New word वेदान्त", allow_growth=True)  # grows vocab
```
